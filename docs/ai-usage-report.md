2. **Learning New Techniques:** Through AI suggestions, I learned about:
   - GSAP animation library and ScrollTrigger
   - CSS custom properties (variables) for theming

3. **Code Quality:** AI helped maintain consistent coding patterns and suggested best practices like semantic HTML and accessibility attributes (`aria-label`).

4. **Problem Solving:** When facing layout issues on mobile devices, AI helped diagnose and fix responsive design problems quickly.

### Challenges
1. **Over-reliance Risk:** Sometimes AI suggestions were overly complex for simple tasks. I had to consciously simplify solutions.

2. **Context Understanding:** AI occasionally suggested code that didn't fit the existing project structure, requiring manual adjustments.

3. **Debugging AI Code:** When AI-generated code had bugs, understanding the underlying logic to fix it required additional effort.

4. **Version Compatibility:** Some suggestions used newer JavaScript features that needed verification for browser compatibility.
